GeminiSharp

A lightweight and easy-to-use C# SDK for the Google Gemini API.

Project Overview

GeminiSharp is a .NET client library designed to simplify interaction with Google's powerful Gemini models. It provides a comprehensive set of tools to support text, chat, image generation, file uploads, and more, making it easy to integrate the Gemini API into your .NET applications.

Key Features

  • Full API coverage for the Google Gemini v1 API.
  • Configurable retry logic using Polly for resilient applications.
  • Structured logging with Serilog for better observability.
  • Helper utilities for JSON schema generation and file-to-base64 conversion.
  • Asynchronous support for all API calls.
  • Easy to configure and get started.
