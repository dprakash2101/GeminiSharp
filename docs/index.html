GeminiSharp SDK

Seamless C# Client for Google Gemini API

About GeminiSharp

GeminiSharp is a C# SDK to simplify integration with Google's Gemini API. Generate text, images, structured outputs, JSON schemas, and more with a clean, robust interface for .NET developers.

GitHub Repo NuGet Package

Core Features

  • Text & Structured Output: Free-form text or JSON based on custom schemas.
  • JSON Schema Helper: Auto-generate JSON schemas from C# classes.
  • Image Generation: Create images from prompts.
  • Image Helpers: Upload via URL, file path, or raw Base64.
  • Retry & Error Handling: Built‑in resilience with clear exceptions.
  • Logging: Serilog integration for detailed SDK logs.
  • Configurable: Models, base URLs, API keys, and timeouts.

JSON Schema Helper

The JsonSchemaHelper class lets you:

  • Convert any C# model into a JSON schema.
  • Validate responses against your schema.
  • Customize schema settings (required fields, defaults, formats).
// Example: Generate schema from a C# class
var schema = JsonSchemaHelper.GenerateSchema<MyCustomModel>();
Console.WriteLine(schema.ToJson());

Image Helpers

Use the built‑in helpers to prepare images for the Gemini API:

  • From File Path: 
    var imgData = ImageHelper.FromFile("path/to/image.png");
  • From URL: 
    var imgData = await ImageHelper.FromUrlAsync("https://example.com/img.jpg");
  • From Base64: 
    var imgData = ImageHelper.FromBase64(base64String);

Installation

Install via NuGet:

dotnet add package GeminiSharp

Supported .NET: 6, 7, 8.

Quick Start

Basic text generation:

using GeminiSharp.Client;

var client = new GeminiClient(httpClient, "API_KEY");
var resp = await client.GenerateContentAsync("gemini-2.0", "Hello, world!");
Console.WriteLine(resp.Candidates[0].Content);
