Features

  • Text & Structured Output: Generate free-form text or structured JSON based on custom schemas.
  • Image Generation: Create visuals from text prompts with ease.
  • Retry Configuration: Handles transient errors automatically for reliable API calls.
  • Unified API Client: Single client for text, image, and structured content.
  • Logging Support: Integrates with Serilog for detailed SDK logs.
  • Flexible Configuration: Customize models, base URLs, and API keys.
  • Robust Error Handling: Clear error messages and exceptions for debugging.

Installation

Install GeminiSharp via NuGet:

dotnet add package GeminiSharp

Supported .NET versions: 6, 7, and 8.

Quick Start

Generate text with a simple API call:

using GeminiSharp.Client;

var apiKey = "your-gemini-api-key";
using var httpClient = new HttpClient();
var geminiClient = new GeminiClient(httpClient, apiKey);

var response = await geminiClient.GenerateContentAsync("gemini-2.0", "Hello, Gemini!");
Console.WriteLine(response?.Candidates?[0].Content);

For image generation and structured output, check the full docs.
