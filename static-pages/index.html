About GeminiSharp

GeminiSharp is a user-friendly C# SDK for .NET developers looking to seamlessly integrate with Google's cutting-edge Gemini API. Whether you aim to generate creative text, produce stunning images, or create structured data outputs, GeminiSharp provides a clean, efficient, and robust interface to unlock the power of Gemini within your .NET projects.

Important Architectural Change: Re-architecting the SDK

We are currently undergoing a significant re-architecture of the GeminiSharp SDK to improve maintainability, modularity, and extensibility. The primary change involves breaking down the monolithic GeminiClient and previously separate clients like GeminiStructuredResponseClient into smaller, specialized client classes, each responsible for a specific area of Gemini API functionality (e.g., TextClient, ImageClient, DocumentClient, AudioClient, VideoClient, StructuredContentClient, UtilityClient).

Key Changes:

  • Specialized Clients: Functionality previously bundled in GeminiClient and other specific clients has been moved to dedicated client classes (e.g., TextClient for text generation, ImageClient for image-related operations).
  • Facade Pattern: The main GeminiClient now acts as a facade, instantiating and delegating calls to these specialized clients. This maintains a consistent public API while allowing for internal modularity.
  • Default Model Update: Where applicable, the default Gemini model has been updated to gemini-2.5-flash for improved performance and access to the latest features.
  • Code Execution Feature (Temporarily Removed): The code execution feature has been temporarily removed during this re-architecture phase to streamline the process. It will be re-introduced in a future update.

This re-architecture aims to make the SDK easier to understand, use, and contribute to, while also paving the way for more robust and feature-rich implementations in the future.

Key Features

  • Specialized Clients: Access Gemini API functionalities through dedicated clients like TextClient, ImageClient, DocumentClient, AudioClient, VideoClient, StructuredContentClient, and UtilityClient.
  • Facade Pattern: The main GeminiClient provides a unified entry point, delegating calls to the appropriate specialized client.
  • Text Generation & Structured Output: Effortlessly generate free-form text responses or structured JSON outputs based on your defined schemas.
  • Image Generation from Text: Unleash your creativity and generate visuals directly from textual prompts with remarkable ease.
  • Robust Automatic Retry Mechanism: Automatically handles transient API errors, ensuring your applications remain resilient and API calls are reliably executed.
  • Integrated Logging with Serilog: Gain deep insights into SDK operations with seamless integration with Serilog for detailed and configurable logs.
  • Highly Flexible Configuration: Tailor the SDK to your needs by customizing models, API base URLs, API keys, and more.
  • Comprehensive Error Handling: Spend less time debugging with clear, informative error messages and exceptions that simplify issue resolution.

Installation

To integrate GeminiSharp into your .NET project, install the NuGet package:

dotnet add package GeminiSharp

GeminiSharp officially supports .NET versions 6, 7, and 8.

Quick Start Guide

Get started generating text in just a few lines of code:

using GeminiSharp.Client;

var apiKey = "YOUR_GEMINI_API_KEY"; // Replace with your actual API key
using var httpClient = new HttpClient();
var geminiClient = new GeminiClient(apiKey, httpClient);

// Using the TextClient via GeminiClient facade
var response = await geminiClient.GenerateTextAsync(null, "Hello, Gemini! What's Falcon 9?");
Console.WriteLine(response?.Candidates?[0].Content?.Parts?[0].Text);

For detailed guidance on image generation, structured output, and advanced configurations, please refer to the complete documentation.
